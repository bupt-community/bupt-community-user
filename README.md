# BUPT-Community-User
User frontend for BUPT-Community.

## Dependencies
* Vue 2
* Ant Design Vue

## Environment Prepare

### Install dependencies

```bash
yarn
```

### Compiles and hot-reloads for development

```bash
yarn serve
```

### Compiles and minifies for production

```bash
yarn build
```
